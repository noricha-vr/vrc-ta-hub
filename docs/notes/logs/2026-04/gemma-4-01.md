## Gemma 4 の常駐モデル候補を整理した
- 日付: 2026-04-08
- 関連: なし
- 状況: `Gemma 4` が正式名称として存在するかを確認したうえで、M3 Ultra / RAM 500GB 環境で常駐させるならどのモデルが妥当かを整理したかった。
- 問題: `Gemma 4` は複数サイズと系統があり、品質重視の `31B`、低レイテンシ寄りの `26B A4B`、軽量な `E2B` / `E4B` のどれを主軸にするか判断材料が散っていた。
- 対応:
  1. Google の `Gemma releases` と Google Blog を確認し、`Gemma 4` が 2026-03-31 / 2026-04-02 時点で正式リリース済みであることを確認した
  2. 公式説明から `31B` は品質寄り、`26B A4B` はレイテンシ寄り、`E2B` / `E4B` はエッジ寄りという位置づけを整理した
  3. M3 Ultra / RAM 500GB なら `gemma-4-31B-it` を第一候補、速度重視なら `gemma-4-26B-A4B-it` を代替候補とする結論にした
- → how/gemma-4.md に知識として追記済み
