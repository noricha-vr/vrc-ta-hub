# スタッフを招待する

集会の運営を手伝ってもらうスタッフを招待する手順を説明します。

**この操作は主催者のみが実行できます。**

## スタッフの役割

スタッフは以下の操作を行えます。

| 操作 | スタッフ | 主催者 |
|------|----------|--------|
| イベントの登録・編集・削除 | OK | OK |
| LT情報の登録・編集・削除 | OK | OK |
| LT申請の承認・却下 | OK | OK |
| 集会情報の編集 | OK | OK |
| スタッフの招待・削除 | NG | OK |
| 主催者の変更 | NG | OK |
| 集会の閉鎖・削除 | NG | OK |

## 招待方法

スタッフを招待するには、**招待リンク**を使用します。

### 手順1: 招待リンクを生成

1. ログインした状態で「マイ集会」ページを開く
2. 「設定」をクリック
3. 「メンバー管理」セクションを開く
4. 「招待リンクを生成」をクリック

### 手順2: リンクを共有

生成された招待リンクを、招待したい相手に共有します。

共有方法の例:
- Discord のDM
- VRChat内で口頭で伝える
- メールで送信

### 手順3: 相手がリンクから参加

1. 招待された人がリンクをクリック
2. ログイン（アカウントがない場合は作成）
3. 招待を承諾

## 招待リンクの仕様

### 有効期限

招待リンクには有効期限があります。

| 項目 | 値 |
|------|-----|
| デフォルト有効期限 | 7日間 |
| 使用回数 | 1回のみ |

期限が切れた場合は、新しいリンクを生成してください。

### セキュリティ

- リンクを知っている人は誰でもスタッフになれます
- 信頼できる相手にのみ共有してください
- リンクの流出に注意してください

## 招待の取り消し

生成した招待リンクを無効にすることもできます。

1. 「メンバー管理」セクションを開く
2. 「招待リンク一覧」で対象のリンクを探す
3. 「無効化」をクリック

## 招待後の確認

スタッフが正常に追加されたかどうかは、メンバー一覧で確認できます。

1. 「設定」→「メンバー管理」
2. メンバー一覧に新しいスタッフが表示されます

## よくある質問

### Q: 招待リンクが機能しない

以下を確認してください。

- リンクの有効期限が切れていないか
- リンクが既に使用済みでないか
- 相手がログインしているか

### Q: スタッフを後から主催者にできますか？

はい、「[主催者を引き継ぐ](/guide/member/transfer/)」の手順で変更できます。

### Q: 招待できる人数に制限はありますか？

いいえ、スタッフの人数に制限はありません。

### Q: 招待した相手がアカウントを持っていない場合は？

招待リンクをクリックすると、アカウント作成ページに誘導されます。Discord連携でアカウントを作成後、自動的にスタッフとして登録されます。
