# 集会を登録する

VRChat技術・学術系イベントHubに新しい集会を登録する手順を説明します。

## 事前準備

集会を登録する前に、以下を準備してください。

- **アカウント**: Discord連携でログイン済みであること
- **集会の基本情報**: 集会名、説明文、カテゴリなど
- **ポスター画像** (推奨): 集会のイメージ画像（1:1の正方形推奨）

## 登録手順

### 手順1: 集会登録ページを開く

1. 画面右上のアカウントメニューをクリック
2. 「集会を追加」を選択

または、マイ集会ページから「新しい集会を登録」をクリックします。

### 手順2: 基本情報を入力する

以下の情報を入力してください。

#### 必須項目

| 項目 | 説明 |
|------|------|
| 集会名 | 集会の正式名称（例: VRChat技術学術勉強会） |
| 説明 | 集会の概要。どんな内容を扱うか、どんな人が参加できるかなど |
| カテゴリ | 集会のジャンル（技術、学術、創作など） |

#### 任意項目

| 項目 | 説明 |
|------|------|
| ポスター画像 | 集会のイメージ画像。一覧ページで表示されます |
| Twitter/X | 集会のTwitter/Xアカウント |
| Discord | 招待リンク |
| VRChatグループ | VRChatグループのURL |
| 参加方法 | 参加の仕方（グループジョイン、フレンド申請など） |

### 手順3: 開催スケジュールを設定する

定期開催の場合は、開催スケジュールを設定できます。

- **曜日**: 毎週何曜日に開催するか
- **時間**: 開催時刻
- **頻度**: 毎週、隔週、月1回など

### 手順4: 内容を確認して登録

入力内容を確認し、「登録」ボタンをクリックして完了です。

## 登録後の流れ

### 承認待ち状態

新規登録された集会は「承認待ち」の状態になります。この状態では、一般公開されません。

### 承認を得る

公開するには、既存の承認済み集会の主催者から承認を得る必要があります。

1. 承認をしてもらいたい集会の主催者に連絡
2. その方が「承認待ち集会一覧」から承認操作を実施
3. 承認されると集会一覧に公開されます

## よくある質問

### Q: 承認がなかなか得られません

技術・学術系のDiscordサーバーやVRChatコミュニティで、承認をお願いしてみてください。同じジャンルの集会の主催者に声をかけるとスムーズです。

### Q: 登録情報を間違えました

登録後でも情報は編集できます。「[集会情報を編集する](/guide/community/edit/)」をご覧ください。

### Q: ポスター画像のサイズは？

推奨サイズは1:1の正方形です。アップロード時に自動でリサイズされます。
