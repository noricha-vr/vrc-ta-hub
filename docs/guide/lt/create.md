# LT情報を登録する

イベントにLT（ライトニングトーク）や発表の情報を登録する手順を説明します。

## LT情報とは

LT情報には以下の内容を登録できます。

- 発表者名
- 発表テーマ
- YouTube動画のURL
- スライド資料
- 関連リンク

## 登録できる人

- **主催者**: すべてのイベントにLT情報を登録可能
- **スタッフ**: すべてのイベントにLT情報を登録可能
- **一般参加者**: LT申請機能が有効な集会で、申請が承認された場合

## 登録手順

### 手順1: イベント詳細ページを開く

1. ログインした状態で「マイ集会」ページを開く
2. LT情報を登録したいイベントをクリック

### 手順2: LT登録フォームを開く

イベント詳細ページで「LT・発表を追加」ボタンをクリックします。

### 手順3: 情報を入力する

#### 必須項目

| 項目 | 説明 |
|------|------|
| 発表者名 | 発表者の名前（VRChat名など） |
| テーマ | 発表のタイトル |

#### 任意項目

| 項目 | 説明 |
|------|------|
| YouTube URL | 発表のアーカイブ動画URL |
| スライドURL | 外部のスライド共有サービスのURL |
| スライドファイル | PDFファイルのアップロード |
| 説明 | 発表内容の詳細説明 |

### 手順4: 確認して登録

入力内容を確認し、「登録」ボタンをクリックして完了です。

## YouTube URLの入力

YouTube動画のURLを入力すると、以下の機能が利用できます。

### 対応するURL形式

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`

### 動画の埋め込み

URLを入力すると、発表詳細ページに動画プレーヤーが表示されます。

### 文字起こしの取得

動画に字幕がある場合、ブログ記事の自動生成に使用されます。

## スライド資料の登録

スライド資料は2つの方法で登録できます。

### 方法1: URLを入力

以下のサービスに対応しています。

- Speaker Deck
- SlideShare
- Google Slides
- その他のスライド共有サービス

### 方法2: PDFをアップロード

PDFファイルを直接アップロードできます。

| 制限事項 | 値 |
|----------|-----|
| 最大ファイルサイズ | 30MB |
| ファイル形式 | PDF |

## 登録後の編集

登録したLT情報は、イベント詳細ページからいつでも編集できます。

1. 対象のLTをクリック
2. 「編集」ボタンをクリック
3. 情報を修正して「保存」

## よくある質問

### Q: 複数のLTを一度に登録できますか？

いいえ、1件ずつ登録してください。登録後、続けて次のLTを追加できます。

### Q: 発表者本人以外が登録してもいいですか？

はい、主催者・スタッフが代理で登録することは問題ありません。

### Q: YouTube動画がまだない場合は？

先にテーマと発表者名だけ登録しておき、後から動画URLを追加することができます。
