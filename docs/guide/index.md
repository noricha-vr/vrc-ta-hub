# はじめに

VRChat技術・学術系イベントHubへようこそ。このガイドでは、集会の主催者やスタッフとしてサイトを活用するための基本的な操作方法を説明します。

## このガイドについて

このガイドは、以下の方を対象としています。

- 新しく集会を登録したい方
- 集会のイベント情報を管理したい主催者・スタッフの方
- LT情報の登録や記事生成機能を活用したい方

## まずやること

### 1. アカウントを作成する

まだアカウントをお持ちでない場合は、**Discord連携**でアカウントを作成してください。

1. 画面右上の「登録」をクリック
2. Discordアカウントでログイン
3. 必要な情報を入力して登録完了

### 2. 集会を登録する

アカウント作成後、自分の集会を登録できます。

1. ログイン後、画面右上のアカウントメニューから「集会を追加」をクリック
2. 集会の基本情報を入力
3. 他の承認済み集会の主催者に承認してもらう

詳しくは「[集会を登録する](/guide/community/create/)」をご覧ください。

### 3. イベントを登録する

集会を登録したら、開催予定のイベントを登録しましょう。

1. イベント一覧ページから「イベント登録」をクリック
2. 開催日時、場所などの情報を入力
3. Googleカレンダーに自動的に反映されます

詳しくは「[イベントを登録する](/guide/event/create/)」をご覧ください。

## 主な機能

### 集会管理

- 集会の基本情報（名前、説明、カテゴリなど）の編集
- ポスター画像のアップロード
- 開催スケジュールの設定
- スタッフの招待と管理

### イベント管理

- イベントの登録・編集・削除
- Googleカレンダーとの連携
- VRChatイベントカレンダーへの投稿サポート

### LT・発表情報管理

- LT情報の登録
- YouTube動画やスライド資料の登録
- AIによるブログ記事の自動生成
- LT申請の承認・却下

### 外部連携

- Discord Webhookによる通知
- LT申請フォームの設定
- Twitterテンプレートによる告知支援

## ヘルプが必要な場合

操作でお困りの場合は、以下の方法でお問い合わせください。

- [GitHubのIssue](https://github.com/noricha-vr/vrc-ta-hub/issues)で報告
- 技術・学術系集会のコミュニティで相談
