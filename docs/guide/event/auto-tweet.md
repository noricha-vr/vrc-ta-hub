# 公式Xアカウントから自動でツイートされる仕組み

VRChat技術・学術系イベントHubでは、登録された情報をもとに公式Xアカウント [@vrc_ta_hub](https://x.com/vrc_ta_hub) から自動で告知ツイートを投稿しています。
**LT・特別回・スライドを登録するだけで、自動で宣伝されるお得な仕組み**です。

## なぜ嬉しいか

- **登録するだけで宣伝が流れる** ―― LT・特別回・スライドを登録すると、Hub の公式アカウントから自動で告知ツイートが投稿されます
- **1回の登録で複数回の露出** ―― 「登録時」「開催当日」「資料公開時」と最大3回ツイートのチャンスがあります
- **集会の知名度UP / 当日のオーディエンス増 / 過去発表の発見性UP** ―― 一度の入力で長く効きます

## 自動ツイートのタイミング一覧

| # | タイミング | きっかけ | 投稿時刻（日本時間） | 主な内容 |
|---|----------|---------|----------|---------|
| 1 | 新規集会公開 | 集会が「承認済み」になった時 | 19:00 | 集会の概要・開催曜日・初回開催日 |
| 2 | LT告知 | LT情報が承認された時 | 19:00 | 登壇者・テーマ・開催日時 |
| 3 | 特別回告知 | 特別回が承認された時 | 19:00 | 同上＋「特別回」明示 |
| 4 | 当日リマインド | 開催日の朝 | 開催日の 19:00 | その日の登壇者・テーマ一覧 |
| 5 | 資料公開 | 終了後にスライドや YouTube を追加した時 | 19:00 | 登壇者・テーマ・公開された資料種別 |

ツイート文は登録後に即時生成されますが、実際の投稿は **毎日 19:00（日本時間）のスケジューラ実行時** にまとめて行われます。
登録直後にXアカウントを見ても投稿されていない場合は、19:00 を過ぎてから確認してください。

## 実際のツイート例

### 1. 新規集会公開時

```
新しい集会「CS集会」が始まりました

毎週月曜 21:00〜
コンピュータサイエンスを楽しく学ぶVRChat集会です

初回開催: 4/14(月) 21:00〜

詳細はこちら https://vrc-ta-hub.com/community/11/
#CS集会
#VRChat技術学術
```

### 2. LT告知

```
4/14(月) 21:00〜 CS集会

上羽未栞さん「April Fools' Day RFC」
エイプリルフールに公開される真面目なジョークRFCの歴史と最新事情を紹介します

技術と遊び心が交差する一夜、ぜひ集会で

詳細はこちら https://vrc-ta-hub.com/community/11/
#CS集会
#VRChat技術学術
```

### 3. 当日リマインド

```
今夜 21:00〜 CS集会

上羽未栞さん「April Fools' Day RFC」
エイプリルフールに公開されるジョークRFCの歴史やユニークな仕様について紐解きます

技術的な遊び心に触れながら楽しい夜を過ごしましょう

詳細はこちら https://vrc-ta-hub.com/community/11/
#CS集会
#VRChat技術学術
```

### 4. 資料公開（過去発表）

```
スライド公開のお知らせ

CS集会で発表された 上羽未栞さん「April Fools' Day RFC」のスライドが公開されました
ジョーク RFC が真面目に書かれてきた歴史を眺めるだけで楽しめる一冊です

詳細はこちら https://vrc-ta-hub.com/event/detail/650/
#CS集会
#VRChat技術学術
```

LT情報を1件登録するだけで、**告知 → 当日リマインド → 資料公開** と複数のタイミングで露出が得られます。

## スライド/YouTube は事後追加でもOK

発表が終わった後でも、スライドURL・YouTube URL・PDFを追加すれば「資料公開ツイート」が自動で流れます。
**過去の発表が再注目されるチャンス**なので、資料が公開できる段階になったら忘れず追加してください。

## ハッシュタグの設定

集会ごとに固有のハッシュタグを設定できます。

### 設定方法

[集会情報を編集する](/community/update/) ページから「Twitterハッシュタグ」フィールドに **`#` も含めて** 入力してください。

### 設定例

| 入力値 | ツイート末尾に挿入される文字列 |
|--------|------------------------------|
| `#CS集会` | `#CS集会` |
| `#VRC_ML集会` | `#VRC_ML集会` |
| 未設定 | （集会固有タグなし） |

設定したハッシュタグに加えて、共通タグ `#VRChat技術学術` が必ず併記されます。
コミュニティ内の検索やトレンドに乗せたい場合は強く推奨します。

## 投稿されるアカウント

**[@vrc_ta_hub](https://x.com/vrc_ta_hub)** から投稿されます。

主催者・登壇者の方は、自動投稿されたツイートを **リツイート / 引用RT で拡散** していただくと、さらに多くの方に届きます。

## 投稿履歴の確認

スタッフ以上のメンバーは、**自分が所属する集会のツイートキュー**（生成中・投稿待ち・投稿済み）を [ツイートキュー一覧](/twitter/queue/) ページから確認できます。
他のコミュニティのキューは表示されません。

スタッフ以上ではない方は、[@vrc_ta_hub](https://x.com/vrc_ta_hub) を直接フォローしてタイムラインをご確認ください。

## よくある質問

### Q: 投稿される内容を事前にプレビューできますか？

スタッフ以上のメンバーは [ツイートキュー一覧](/twitter/queue/) で投稿前の文面を確認できます（編集は運営のみが可能）。
スタッフ以上の権限がない方は、テーマや説明文を充実させることで品質をコントロールしてください。

### Q: 同じLTが何度もツイートされますか？

役割の違うタイミングで複数回ツイートされます（告知/当日リマインド/資料公開）。
それぞれ目的が異なり、文面も AI によって変化するため、フォロワーから見ても重複感は少なく仕上がります。

### Q: ツイートされませんでした。なぜ？

以下を順にご確認ください。

1. LT/特別回が **「承認済み」** になっているか（申請ベースのLTは承認操作が必要）
2. 開催当日の場合は **19:00（日本時間）以降** に [@vrc_ta_hub](https://x.com/vrc_ta_hub) を確認
3. 集会自体のステータスが「承認済み」か
4. それでも投稿されない場合は運営までご連絡ください

### Q: 投稿された後に内容を修正したい

X上での編集は不可（削除→再投稿が必要）です。
当日リマインドの場合は、開催当日 19:00 のバッチで再生成されるので、それまでに登録情報を修正しておけば反映されます。

### Q: 自動投稿を止めたい / 投稿されたくない発表がある

主催者から個別に停止する機能は現状ありません。投稿を停止したい場合は運営までご連絡ください。
