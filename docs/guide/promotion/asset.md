# アセットを設置する

あなたのワールドでも、VRChat技術・学術系イベントHubの開催予定を一覧表示できます。

## 技術・学術系イベント情報アセット

技術・学術系集会に特化した開催予定共有アセットを無料配布しています。

### このアセットでできること

- ワールド内に技術・学術系イベントの開催予定を一覧表示できます
- VCC から導入して Prefab を置くだけで設置できます
- 元データの JSON が更新されると表示内容も自動で更新されるため、設置後の更新負担がほとんどありません
